# Bank Marketing Data Analysis and Modeling

This project is a homework assignment for ADS 2026 based on the Bank Marketing dataset.  
It includes data analysis, preprocessing and feature engineering.

## Author
**Reza Kazemeiny Moghaddam**

## Dataset
This project uses the following dataset:

- Kaggle link: [Bank Marketing Dataset](https://www.kaggle.com/datasets/henriqueyamahata/bank-marketing)

## Project Description
In this project, the first three sections of the PDF assignment have been completed in code.  
The notebook contains the related implementation, analysis, and results.

The workflow is based on the assumption that the dataset is already prepared for training and modeling.  
Accordingly, the data is treated as ready for machine learning tasks after the necessary preprocessing steps.

## Main Steps
The notebook includes the following tasks:

1. Data loading and initial inspection
2. Data cleaning and preprocessing
3. Exploratory data analysis
4. Feature engineering
5. Feature importance analysis
6. Model preparation and evaluation

## Notes on Results
The reported results may vary slightly across different runs because of randomness in the process, such as data splitting or model initialization.  
However, the overall conclusions and patterns remain consistent, and the results are reported in a general and stable way.
